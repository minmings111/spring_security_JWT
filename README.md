**# Mini Mings**

**>** JWT 기반 사용자 인증 시스템

**## 📌 프로젝트 소개**

Spring Boot와 JWT를 활용한 안전한 사용자 인증 시스템입니다.

**## 🏗 프로젝트 구조**

**-\*\***backend/\*\* - Spring Boot 백엔드

** -** JWT 인증 시스템

** -** Spring Security 설정

** -** MyBatis + MySQL

**-\*\***frontend/\*\* - 프론트엔드 (개발 예정)

**## ✨ 주요 기능**

**### ✅ 현재 구현 상태**

**- JWT 인증 인프라 구축**

**- Spring Security 기본 통합**

**- 데이터베이스 스키마 설계 (소셜 로그인 대비)**

**- 인증이 필요한 경로에 대한 기본 접근 제어 설정**

**### 🚧 개발 예정**

**- 회원가입/로그인 API 구현**

**- 웹 프론트엔드 연동**

**- OAuth 2.0 소셜 로그인 (Google, Kakao)**

**- 사용자 프로필 관리**

**## 🛠 기술 스택**

\***\*Backend\*\***

**- Java 21**

**- Spring Boot 4.0.1**

**- Spring Security + JWT**

**- MyBatis**

**- MySQL 8.0**

\***\*Frontend** (예정)\*\*

**- React / Next.js**

**- TypeScript**
